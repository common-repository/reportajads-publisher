Get reportage from the Reportaj ADS panel and publish it on your website.

== Description ==

<p>The Reportaj Ads Publisher plugin is a robust tool designed to work seamlessly with the Reportaj Ads platform, a third-party service that empowers you to create, manage, and publish various types of content on your website. This plugin allows you to easily prepare and publish guest posts, review posts, suggested posts, and advertorial content.</p>

<p><a href="https://reportaj.ir/">Reportaj Ads</a> is the third-party platform that the Reportaj Ads Publisher plugin relies on. It offers a user-friendly interface for content creation and provides strong tools for developing and managing diverse content types.</p>

== Installation ==

1. Upload the plugin files to the `wp-content/plugins/reportajads-publisher` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the `Plugins` screen in WordPress.
3. Use the Settings -> Reportaj ADS Publisher screen to configure the plugin.
4. Follow any additional instructions provided to successfully integrate the plugin with your Reportaj ADS account.

== Frequently Asked Questions ==

= How do I use this plugin? =

After installing the plugin, you need to configure the connection to the Reportaj ADS panel by entering the Web Service Code and Site Code. These settings can be found under the "Reportaj ADS Settings" submenu, which is visible in the WordPress settings. Ensure that these codes are correctly set up to enable the plugin to function properly and fetch reportage ads from your account.

= How do I get a Reportaj ADS account? =

To use this plugin, you need to create an account at Reportaj ADS. Visit their website and follow the registration process.
